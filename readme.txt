Displays prayer times for mosques as a lightweight widget. Fully translatable!

== Description ==

The Owlish Masjid Prayer Times Widget displays the times for the 5 daily prayers. The time for Fajr, Dhuhr, Assr, Maghrib and Isha can easily be set in the WordPress Dashboard. Just install and activate the plugin, drag it to your widgets and enter the current times when prayer is offered in the mosque.

**Usage**

Install the plugin as described in Tab "Installation". Drag the widget to your sidebar and enter the times directly in the widget.

**Features**

* Lightweight free plugin to display the prayer times for the masjid
* Fully translatable Widget + frontend translation directly in the backend


== Installation ==

1. Upload `owlish-masjidprayertimes.zip` to the `/wp-content/plugins/` directory or use the "Add new" dialogue on the WordPress backend.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place the widget via the appearance menu in your WordPress admin interface.
4. Enter the times the prayers are offered directly in the widget.

== Frequently Asked Questions ==

= Does this plugin display the times automatically? =

No. This plugin displays the times set in the backend. It is best for any mosque whose times does not change a lot. For individual prayer times you can install the <a href="https://wordpress.org/plugins/owlish-prayer-times/">"Owlish Prayer Times"</a>.
